## 0. Setting and conventions

p any prime; Q := p². φ = x² + sx + t ∈ ℤ_p[x] monic with φ̄ irreducible over F_p
(irreducibility used only to make F_Q := F_p[x]/(φ̄) a field; the bijection algebra never
uses it). Every monic quartic f has the unique φ-adic development f = φ² + a₁φ + a₀ with
deg aᵢ ≤ 1; identify a deg-≤1 polynomial u₁x + u₀ with the coordinate pair (u₁, u₀) ∈ ℤ_p².
The **height-h digit** of a pair is its coefficientwise p-adic digit pair, read as the
element of F_Q via the basis (x̄, 1); the **Gauss valuation** v(a) = min(v_p(u₁), v_p(u₀)).
D1m (proved: blueprint W8 §1 + Lean `OM/DevBijection.lean`): (c₃,…,c₀) ↦ (a₁, a₀) is
affine ℤ[s,t]-unitriangular, hence exact-count on every digit cylinder.

**The descend stratum** at slope m ≥ 1 with center c̃ ∈ F_Qˣ, inside the root-cluster
ambient A := {digit₀(a₁) = digit₀(a₀) = 0}:

  S_{m,c̃} := { (a₁, a₀) : v(a₀) ≥ 2m, digit_{2m}(a₀) = c̃²,
                v(a₁) ≥ m,  digit_m(a₁)  = −2c̃ }.

(At p = 2, −2c̃ = 0, recovering the char-2 "square residual ⟺ v(a₁) ≥ m+1" form.) The
residual polynomial of the slope-−m side is y² + digit_m(a₁)·y + digit_{2m}(a₀) =
(y − c̃)² exactly on S_{m,c̃} — the μ = 2 descend read.

**The child key** (the corrected sign, §5.1 of blueprint W8): ĉ := the minimal-support
lift of c̃ (deg ≤ 1, coefficients Teichmüller/minimal — any FIXED lift with ĉ̄ = c̃ works
for everything below), and

  φ' := φ − p^m·ĉ .

φ' is again monic quadratic, φ̄' = φ̄. Re-develop: f = φ'² + a₁'φ' + a₀', deg aᵢ' ≤ 1.

## 1. Lemma J1a (the descend re-development map, explicitly)

Write ĉ = ĉ₁x + ĉ₀, a₁ = a₁₁x + a₁₀, φ' = x² + s'x + t' with s' = s − p^mĉ₁,
t' = t − p^mĉ₀. Substituting φ = φ' + p^mĉ into f = φ² + a₁φ + a₀ and reducing the two
deg-2 products by the monic φ' (single division steps: for deg-≤1 u, w one has
u·w = (u₁w₁)·φ' + [(u₁w₀ + u₀w₁ − u₁w₁s')x + (u₀w₀ − u₁w₁t')]):

  Θ_ĉ(a₁, a₀) = (a₁', a₀') with
  a₁'₁ = a₁₁ + 2p^mĉ₁
  a₁'₀ = a₁₀ + 2p^mĉ₀ + p^m·ĉ₁a₁₁ + p^{2m}·ĉ₁²
  a₀'₁ = a₀₁ + p^m[ĉ₁a₁₀ + ĉ₀a₁₁ − ĉ₁a₁₁s'] + p^{2m}[2ĉ₁ĉ₀ − ĉ₁²s']
  a₀'₀ = a₀₀ + p^m[ĉ₀a₁₀ − ĉ₁a₁₁t'] + p^{2m}[ĉ₀² − ĉ₁²t']

Θ_ĉ is affine over ℤ[s,t,ĉ₁,ĉ₀] with lower-unitriangular linear part in the order
(a₁₁, a₁₀, a₀₁, a₀₀), and **Θ_ĉ ≡ id mod p^m** (every non-identity term carries p^m).
Hence Θ_ĉ is a bijection of ℤ_p⁴ and of (ℤ/p^N)⁴ for every N, commuting with reduction.

*Proof.* The substitution identity and the two reduction steps are the displayed algebra;
(φ' + p^mĉ)² = φ'² + 2p^mĉφ' + p^{2m}ĉ², a₁(φ' + p^mĉ) = a₁φ' + p^mĉa₁, and the two
products ĉ², ĉa₁ reduce by the displayed single-step formula. Triangularity and the p^m
factor are visible; unitriangular ⟹ determinant 1 ⟹ bijectivity at every level;
ℤ-polynomial coefficients ⟹ reduction-compatibility. ∎

*(Machine anchor: the four formulas satisfy the defining identity f = φ'² + a₁'φ' + a₀'
identically — checkable by `ring`; empirically exercised by the three census gates.)*

## 2. Lemma J1b (staircase inclusion: the child lands in the child cluster box)

On S_{m,c̃}: v(a₁') ≥ m + 1 and v(a₀') ≥ 2m + 1.

*Proof.* Heights < m of a₁' equal those of a₁ (Θ ≡ id mod p^m), all zero. Height m:
digit_m(a₁') = digit_m(a₁) + 2c̃ + [digit₀(ĉ₁a₁₁-term)·p^{m}-shift: v(a₁₁) ≥ m gives that
term v ≥ 2m ≥ m+1, invisible at height m] = −2c̃ + 2c̃ = 0. For a₀': heights < 2m get
contributions only from a₀ (zero: v(a₀) ≥ 2m) and from p^m·(ĉa₁-remainder) whose Gauss
valuation is ≥ m + v(a₁) ≥ 2m, and p^{2m}-terms — all invisible below 2m. Height 2m, read
in F_Q (the remainder-mod-φ' reduction at the residue level is multiplication in F_Q since
φ̄' = φ̄): digit_{2m}(a₀') = digit_{2m}(a₀) + c̃·digit_m(a₁) + c̃² = c̃² − 2c̃² + c̃² = 0. ∎

## 3. Theorem J1-step (the descend bijection)

Θ_ĉ restricts to a BIJECTION

  S_{m,c̃}  ⟶  B' := { (a₁', a₀') : v(a₁') ≥ m+1, v(a₀') ≥ 2m+1 }

at every finite level N ≥ 2m + 2 (and hence over ℤ_p).

*Proof.* Inclusion is J1b. Injectivity is J1a. Cardinalities at level N: S_{m,c̃} pins
heights 0..m of a₁ (m+1 digits) and 0..2m of a₀ (2m+1 digits), leaving
Q^{N−1−m}·Q^{N−1−2m}; B' pins heights 0..m of a₁' and 0..2m of a₀' to zero — the same
count. A cardinality-preserving injection between equinumerous finite sets is onto. ∎

**Corollary J1-uniform.** For every digit-cylinder subset C' ⊆ B' (at any level
N ≥ 2m+1): #(S_{m,c̃} ∩ Θ_ĉ⁻¹ C') = #C' — counts, no normalization needed (J1-step is a
bijection S_{m,c̃} → B', so intersecting with the preimage of C' ⊆ B' bijects onto C').
In proportion form: the child digit vector is exactly uniform on S_{m,c̃} with each fiber
of size #S_{m,c̃}/#B'-window. *(This is the statement Cases F / F-odd verify
bucket-by-bucket.)*

## 4. Theorem JOINT-D1 (deg-2-key descend towers, ABSOLUTE heights)

*(Rewritten after Codex pass 1, which found the original §4 stated "slopes" that were
increments — the polygon stays anchored at (2, 0), so heights accumulate ABSOLUTELY and
there is no rescale — and an induction that was gestured, not proved. Both criticals are
repaired here; the one-step §§1–3 were confirmed as proved.)*

**Data.** Depth k ≥ 1; cumulative heights 0 = M₀ < M₁ < ⋯ < M_k (increments
m_j := M_j − M_{j−1} ≥ 1); centers c̃_j ∈ F_Qˣ with fixed minimal lifts ĉ_j. Keys:
φ⁽⁰⁾ := φ, φ⁽ʲ⁾ := φ⁽ʲ⁻¹⁾ − p^{M_j}·ĉ_j. Coordinate maps: Θ_j := the J1a map at
parameters (m := M_j, ĉ := ĉ_j), taking φ⁽ʲ⁻¹⁾-development pairs to φ⁽ʲ⁾-development
pairs. **The j-th read is a Newton read of the CURRENT development polygon anchored at
(2, 0): its descend side has absolute slope −M_j** (constant-degree-2 square residual at
absolute heights M_j, 2M_j); m_j is only the increment of pinned digits.

**Boxes and read strata (all in absolute heights, level N ≥ 2M_k + 2):**
  B_j := { v(a₁) ≥ M_j + 1, v(a₀) ≥ 2M_j + 1 }   (in φ⁽ʲ⁾-coordinates; B₀ = the root
        cluster {v ≥ 1} at M₀ = 0),
  T_j := S_{M_j, c̃_j}   (§0's descend stratum at parameter M_j, in φ⁽ʲ⁻¹⁾-coordinates).
Note T_j ⊆ B_{j−1} automatically (v(a₀) ≥ 2M_j ≥ 2M_{j−1}+1, v(a₁) ≥ M_j ≥ M_{j−1}+1,
using m_j ≥ 1), and J1-step (§3, applied verbatim at m := M_j) says: **Θ_j : T_j → B_j is
a bijection.**

**The joint stratum.** Ψ_j := Θ_j ∘ ⋯ ∘ Θ₁ (a composition of affine unitriangular
bijections, hence such a bijection; Ψ₀ := id). In root-development coordinates:
  J_k := T₁ ∩ Ψ₁⁻¹(T₂) ∩ Ψ₂⁻¹(T₃) ∩ ⋯ ∩ Ψ_{k−1}⁻¹(T_k).
A **leaf condition** is any subset C ⊆ B_k of the φ⁽ᵏ⁾-coordinates (in applications a
digit cylinder above the pinned block, but the theorem needs no structure on C).

**Theorem.** Ψ_k restricts to a bijection J_k → B_k. Consequently, for every leaf
condition C ⊆ B_k:
  #{ development pairs in J_k ∩ Ψ_k⁻¹(C) } = #C,                            (J-D1)
and composing with the D1m development bijection (blueprint W8 §1), the same count holds
for monic quartics f mod p^N in the joint stratum. In particular, with #-values at level
N: #J_k = #B_k = Q^{2N − 3M_k − 2}, the per-step conditional volume of read j inside
B_{j−1} is Q^{−3m_j} per center (fresh pinned digits: heights M_{j−1}+1..M_j of a₁ and
2M_{j−1}+1..2M_j of a₀, the last of each pinned to the center data, the rest to 0 — note
these are exactly the "increments", and the product ∏_j Q^{−3m_j} = Q^{−3M_k} telescopes
consistently with #J_k/#B₀), and any cylinder leaf contributes its own nominal count —
the class-3 joint ledger.

*Proof.* Induction on k. k = 1: J₁ = T₁ and Θ₁ : T₁ → B₁ is J1-step at M₁. Step: assume
Ψ_{k−1} : J_{k−1} → B_{k−1} is a bijection. Then
  J_k = J_{k−1} ∩ Ψ_{k−1}⁻¹(T_k),
and since T_k ⊆ B_{k−1}, the restriction Ψ_{k−1} : J_k → T_k is a bijection (the
preimage-of-subset restriction of a bijection). Composing with Θ_k : T_k → B_k (J1-step
at M_k) gives Ψ_k : J_k → B_k bijective. The count (J-D1) is the same restriction
argument applied to C ⊆ B_k. The volume statements are direct digit counts of T_j inside
B_{j−1} (m_j + 2m_j fresh pinned heights) and of B_k (M_k + 1 + 2M_k + 1 pinned heights
at level N). ∎

**Consistency checks.** (i) Case F's child ladder: after step 1 at M₁ = 1 (p = 2), the
members continuing to a second descend need digit₂(a₁') = 0 and digits 3 zero + digit 4 =
c̃₂² of a₀' — absolute read at M₂ = 2, increment m₂ = 1, conditional volume 4⁻³ per
center: 3072·(3/64) = 144 = the observed (v₀' = 4, v₁' ≥ 3) square-residual bucket count
(432 + 108 + 27 + 9 = 576 total v₀'=4 members, of which the v₁' ≥ 3 slice 144 ✓ matches
the distribution table). (ii) The odd members (v₀' = 2M_{j}+1-type exact) are the child
RAM reads (absolute slope −(2M_j+1)/2, half-integer — leaf, not descend), the geometric
ratio-4 ladder in the table. ∎-check.

/-- **M1 (`ax:montes`): Montes count-multiplicativity over the OM tree, in the COUNTING model.**

A `MontesData q n M` certifies, for the counting model `M : CountingModel q n` (degree `n`, residue

`PadicMeasure` axioms.

The fields encode `ax:partition`, `ax:multiplicativity` (as a PURE-COUNT factorization), `ax:dichotomy`. -/
structure MontesData (q n : ‚Ñï) (M : CountingModel q n) where

  shapesOf : FactorizationType ‚Üí Finset ClusterShape

  order-`r` Newton-polygon face √ó residual-shape stratum, each carrying its `T_BB3` residual count
  data (`dS`, `Œ¥`), `T_BB1` box (`polygon`) and descent `children`. -/
  cells : ClusterShape ‚Üí List CountCell
  /-- **STRICT DESCENT (`ax:dichotomy`, count shadow of `PadicMeasure.descend_size_lt`).** Off the
  self-loop, every descent child of every cell has STRICTLY SMALLER cluster `treeSize`. This is the
  well-foundedness input of the `MontesData.C` recursion. -/
  cells_descend : ‚àÄ (T : ClusterShape), ‚àÄ c ‚àà cells T, ‚àÄ ch ‚àà c.children, treeSize ch < treeSize T
  /-- The level-`N` stratum count `# S_T^{(N)}` of a shape `T` (decided-at-`N` monic `f` realizing the
  combinatorial shape `T`). A `‚Ñö`-valued integer count. -/

  (`boxHaarEquidist`) and the box-additive node-limit (`nodeMeasure_boxSum`) are stated. It is the only
  place the abstract Haar measure enters; the box-wise refactor (2026-06-21) isolates it here. -/

  /-- **THE MINIMAL IRREDUCIBLE BOX-WISE AXIOM (box-Haar + residual-equidistribution).** The normalized
  per-box Haar volume of every cell `c` EQUALS the PROVED box-volume √ó residual-count value
  `countCellCoeff c q = (q^Œ¥)^{dS-1} ¬∑ bb1Value c.polygon q`:

  This is the GENUINE measure-wall content and the ONLY irreducible measure assumption left: the RHS is

  factors ‚Äî it is the per-box Haar-existence + equidistribution statement, box-localized. -/
  boxHaarEquidist : ‚àÄ (c : CountCell) (q' : ‚Ñï), 1 < q' ‚Üí boxMeasure c q' = countCellCoeff c q'

  normalized stratum measure `lim_N stratumCount T N / q^{nN}` EXISTS and is the box-additive SUM over

  undecided_le_split : ‚àÄ N,

  sepTail_tendsto :

/-- **The per-shape coefficient `C_T(q)` ‚Äî the pure-counting per-node factorization.**

`D.C` is the standalone `clusterCount` recursion run over `D`'s own count data
(`cells`/`treeSize`/`cells_descend`): by well-founded recursion on `treeSize` it is the faithful
eq-(4.2) count factorization `(T_BB3 residual count) √ó (T_BB1 box count) √ó ‚àè_children`, the self-loop

Built ENTIRELY from counting blocks ‚Äî NO `PadicMeasure.clusterMeasure`, NO `AX_cellRecursion`. Its

By M1's `nodeMultiplicativity`, this IS the genuine `lim_N stratumCount T N / q^{nN}`. -/

/-- **(2) PER-NODE COUNT-MULTIPLICATIVITY ‚Äî now a THEOREM (the box-wise derivation).**

The normalized stratum measure `lim_N stratumCount T N / q^{nN}` equals the per-node `clusterCount`
factorization ‚Äî DERIVED from the box-wise primitives, no longer a single carried field:

* `nodeMeasure_boxSum` gives the limit = `(Œ£_{cells c} boxMeasure c q ¬∑ ‚àè_children) / countPivot` ‚Äî
  the box-additivity of the node measure (measure existence + additive assembly, the minimal axiom);
* `boxHaarEquidist` rewrites each per-box Haar measure `boxMeasure c q = countCellCoeff c q` ‚Äî the
  proved box-volume `bb1Value` (`T_BB1`) √ó proved residual count `(q^Œ¥)^{dS-1}` (`T_BB3`, residualBoxCount);
* `clusterCount_boxSum` (PROVED box-additivity arithmetic) recombines into `clusterCount`.

So this is now PROVED from {box-volume(proved) √ó residual-count(proved) √ó box-additivity(proved) √ó
box-Haar/equidistribution(minimal axiom)}, exactly the box-wise decomposition. -/
theorem nodeMultiplicativity (T : ClusterShape) :
    Filter.Tendsto (fun N => D.stratumCount T N / (q : ‚Ñö) ^ (n * N))
      Filter.atTop (nhds (clusterCount D.cells D.treeSize D.cells_descend T q)) := by
  have hq : 1 < q := lt_of_lt_of_le Nat.one_lt_two M.hq
  -- the box-additive node limit (minimal axiom):
  have hbox := D.nodeMeasure_boxSum T
  -- the box-additive sum (with per-box = box-volume √ó residual-count) recombines into `clusterCount`:
  have hrecomb :
      (((D.cells T).map (fun c =>
            D.boxMeasure c q
              * (c.children.map (fun ch => clusterCount D.cells D.treeSize D.cells_descend ch q)).prod)).sum
          / countPivot (D.treeSize T) q)
        = clusterCount D.cells D.treeSize D.cells_descend T q := by
    -- rewrite each per-box Haar measure to the proved box-volume √ó residual-count value, then fold by
    -- the PROVED box-additivity `clusterCount_boxSum`.
    rw [clusterCount_boxSum D.cells D.treeSize D.cells_descend T q
      (fun c => D.boxMeasure c q
        * (c.children.map (fun ch => clusterCount D.cells D.treeSize D.cells_descend ch q)).prod)
      (fun c _ => by rw [D.boxHaarEquidist c q hq])]
  rwa [hrecomb] at hbox

/-- **(A1) The discriminant tail tends to `0`** ‚Äî PROVED from the elementary `q^{-N}` decay envelope.
The discriminant-tail measure is squeezed between `0` (`discriminantTailMeasure_nonneg`) and

(`tendsto_pow_atTop_nhds_zero_of_lt_one`). PROVED. -/
theorem discriminantTail_tendsto :
    Filter.Tendsto D.discriminantTailMeasure Filter.atTop (nhds 0) := by
  -- the envelope `discriminantTailConst ¬∑ (q^N)‚Åª¬π` tends to 0
  have hq2 : (2 : ‚Ñö) ‚â§ (q : ‚Ñö) := by exact_mod_cast M.hq
  have hqpos : (0 : ‚Ñö) < (q : ‚Ñö) := by linarith
  -- `(q^N)‚Åª¬π = (q‚Åª¬π)^N` and `0 ‚â§ q‚Åª¬π < 1`
  have hinv_lt : (q : ‚Ñö)‚Åª¬π < 1 := by
    rw [inv_lt_one_iff‚ÇÄ]; right; linarith
  have hinv_nonneg : (0 : ‚Ñö) ‚â§ (q : ‚Ñö)‚Åª¬π := le_of_lt (inv_pos.mpr hqpos)
  have hpow : Filter.Tendsto (fun N => ((q : ‚Ñö)‚Åª¬π) ^ N) Filter.atTop (nhds 0) :=
    tendsto_pow_atTop_nhds_zero_of_lt_one hinv_nonneg hinv_lt
  have henv : Filter.Tendsto (fun N => D.discriminantTailConst * ((q : ‚Ñö) ^ N)‚Åª¬π)
      Filter.atTop (nhds 0) := by
    have h0 : Filter.Tendsto (fun N => D.discriminantTailConst * ((q : ‚Ñö)‚Åª¬π) ^ N)
        Filter.atTop (nhds (D.discriminantTailConst * 0)) :=
      tendsto_const_nhds.mul hpow
    rw [mul_zero] at h0
    refine h0.congr (fun N => ?_)
    rw [inv_pow]
  -- squeeze: 0 ‚â§ discriminantTailMeasure N ‚â§ envelope
  exact tendsto_of_tendsto_of_tendsto_of_le_of_le' tendsto_const_nhds henv
    (Filter.Eventually.of_forall (fun N => D.discriminantTailMeasure_nonneg N))
    (Filter.Eventually.of_forall (fun N => D.discriminantTail_envelope N))

/-- **DECOMPOSITION THEOREM (`thm:decomp`): `countingDensity œÉ = ‚àë_{T ‚àà shapesOf œÉ} C_T(q)`.**

The DECIDED-LIMIT density is the finite sum over the shape menu of the per-shape coefficients. PROVED

theorem coeff_isRational (T : ClusterShape) : IsRationalFn (D.coeff T) := by
  have hgen := OMInduction.clusterVol_isRational
    (Shape := ClusterShape)
    (size := D.treeSize)
    (clusterVol := D.C)
    (contribs := fun T => (D.cells T).map (fun c => (countCellCoeff c, c.children)))
    (pivot := fun T => countPivot (D.treeSize T))
    ?hrec ?hdescend ?hcoeff ?hpivot_rat ?hpivot_ne T
  ¬∑ -- transport `IsRationalFn (D.C T)` to `IsRationalFn (D.coeff T)` (definitionally equal)
    exact hgen
  case hrec =>
    -- the `C_rec` recursion-unfold, with `contribs`/`countCellCoeff` rewritten into engine shape.
    intro T q hq
    rw [D.C_rec T q]
    congr 1            -- peel `_ / countPivot _ q`
    congr 1            -- peel `(_).sum`
    -- the cell-map equals the contribution-map: fuse `(g ‚àò (c ‚Ü¶ (coeff, children)))` and match.
    rw [List.map_map]
    apply List.map_congr_left
    intro c _
    simp only [Function.comp_apply]
  case hdescend =>
    -- every contribution child is a cell child, strictly smaller by `cells_descend`.
    intro T c hc ch hch
    simp only [List.mem_map] at hc
    obtain ‚ü®cell, hcell_mem, hc_eq‚ü© := hc
    rw [‚Üê hc_eq] at hch
    exact D.cells_descend T cell hcell_mem ch hch

    -- each contribution coefficient is `countCellCoeff cell`, rational.

    simp only [List.mem_map] at hc
    obtain ‚ü®cell, _, hc_eq‚ü© := hc
    rw [‚Üê hc_eq]

  case hpivot_rat =>
    intro T
    exact countPivot_isRational (D.treeSize T)
  case hpivot_ne =>
    intro T q hq
    exact countPivot_ne (D.treeSize T) q hq

(`RatFn.isRationalFn_listSum`). PROVED modulo M1, with the only-Montes footprint.

Honest scope: this proves rationality of the function `q ‚Ü¶ ‚àë_T coeff T q` AS A FUNCTION OF `q`; the

    IsRationalFn (fun q' => ‚àë T ‚àà D.shapesOf œÉ, D.coeff T q') := by
  classical
  set L : List (‚Ñï ‚Üí ‚Ñö) := (D.shapesOf œÉ).toList.map (fun T => D.coeff T) with hL

    exact D.coeff_isRational T

    RatFn.isRationalFn_listSum L hLrat
  have heq : (fun q' => (L.map (fun f => f q')).sum)
      = (fun q' => ‚àë T ‚àà D.shapesOf œÉ, D.coeff T q') := by
    funext q'
    rw [hL, List.map_map]
    exact Finset.sum_map_toList (D.shapesOf œÉ) (fun T => D.coeff T q')
  rwa [heq] at hlist
